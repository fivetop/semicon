## 문서 목록

- `folder_structure_spec.md`
  - 디렉터리 구성, 각 폴더의 역할, 실행 버전 분리 방식을 정리한다.
- `execution_guide.md`
  - 실제 실행 절차, 사전 준비 사항, 실행 후 확인 항목을 설명한다.
- `script_reference.md`
  - 셸 래퍼와 각 TCL 스크립트의 역할을 단계별로 설명한다.
- `flow_diagram.md`
