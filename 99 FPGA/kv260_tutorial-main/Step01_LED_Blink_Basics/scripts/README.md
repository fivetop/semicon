# Step01_LED_Blink_Basics - Vivado TCL Scripts

## TCL Script Files

This directory contains TCL scripts for automated Vivado project creation and build.

### create_project.tcl
Creates the Vivado project with KV260 board.

### create_block_design.tcl
Creates the Block Design with Zynq PS and connects the RTL module.

### run_synthesis.tcl
Runs synthesis.

### run_implementation.tcl
Runs implementation and generates bitstream.

### run_all.tcl
Runs all steps in sequence.
